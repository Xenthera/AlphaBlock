# AlphaBlock

A voxel engine game 

### Features
* Occluded faces
* Texture atlas support
* Chunk Support
* Multiple Blocks Implemented
* Custom Camera 
* Player

### Planned
* Chunk Manager
* AABB Physics for Player
* Better Terrain Generation
